# testizi - Easy Test for Node/Javascript

Parse JSDOC of functions and generates tests for them. 

## For functions simple functions

/** 
  @test sum(1, 1) == 2
  @test sum(1, 1) > 1  
*/

function sum(a,b){
  return a+b;
}
